## Getting Started With a Project

Each project includes a friendly README and starter code. To run a project:

```bash
cd project-X/starter
npm install
npm run dev
```

Use the `solution` folder if you want to see a fully completed version or compare your work.

## What You’ll Learn

* Core Vue concepts (reactivity, components, events)
* Routing and navigation with Vue Router
* State management with Vuex or Pinia
* Fetching data from APIs
* Real-world app patterns
* Building from small apps to larger, full-featured projects

## Tips & Notes

* No pressure to complete everything at once—go at your own pace.
* Each project’s README gives guidance and suggestions, screenshots, or GIFs to help you understand what to build.
* Live demos can be linked if available, but focus on learning concepts first.
* Topics for discoverability: `vue`, `vue3`, `learning`, `project-based-learning`.

This repo is made to make learning Vue **fun, approachable, and practical**. Enjoy exploring, building, and experimenting!
